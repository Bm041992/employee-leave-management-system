# Employee & Leave Management System

---

## Features

### Employee Module
- Add Employee
- Edit Employee
- Delete Employee
- List Employees

**Fields:**
- Name
- Email
- Phone
- Department

---

### Leave Module
- Apply for Leave

**Fields:**
- Employee
- From Date
- To Date
- Reason
- Status

---

### Admin Module
- Approve Leave Requests
- Reject Leave Requests

---

## Technologies Used

- Laravel
- PHP
- MySQL
- Bootstrap
- jQuery / AJAX
